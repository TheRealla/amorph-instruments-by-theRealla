# Amorph Instruments by theRealla

Custom patches/code for the free Amorph Instrument plugin (https://artistsindsp.gumroad.com/l/amorph).

## WobbleMorphBass.cmajor
Deep wobbling sub-bass: pure sine low-end that morphs to gritty metallic FM growls/scrapes above C3.

**How to use**:
1. Install Amorph Instrument in your DAW.
2. Load on a MIDI track.
3. Paste the entire contents of WobbleMorphBass.cmajor into the code editor.
4. Hit Compile.
5. Play low notes for clean sub, higher for grit. Tweak knobs: Wobble Rate, Morph Amount, Distortion Drive, Attack/Release.

Parameters are velocity-sensitive and polyphonic (8 voices).
